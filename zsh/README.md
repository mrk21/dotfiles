# zsh

```bash
ln -s $(pwd)/zprofile.sh ~/.zprofile
ln -s $(pwd)/zshrc.sh ~/.zshrc 
ln -s $(pwd)/zprofile.d ~/.zprofile.d
ln -s $(pwd)/zshrc.d ~/.zshrc.d
```

## Dependencies

- coreutils brew
- solarized
